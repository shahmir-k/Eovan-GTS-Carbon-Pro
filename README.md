# Eovan-GTS-Carbon-Pro

My daily driver Electric skateboard build based on Carbon Pro deck upgraded with Carbon Super electronics

- Carbon Super Motors
- Cloudwheels
- Carbon Super V2 ESC
- Shredlights
- Eovan front toe guard handle
